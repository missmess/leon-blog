社交百科

应用项目

阅读5分钟

2022年12月03日 21:11

# 跨平台# Flutter# dart# Android端# iOS端

社交百科是一个知识答题类的网赚app。

主要功能

  • 回答社交领域相关的问题，答对得金币奖励，答错无奖励。连对奖励更多。

  • 看广告得金币，保持连对数据。

  • 到达一定金币数量可以提现。

  • 可以邀请好友，好友收益也可以返利。

  • 包含开屏、插屏、激励视频广告。

我的职责

  1. 负责对标同类型产品设计功能。

  2. 负责UI设计。

  3. 负责flutter应用开发。

  4. 负责广告sdk的接入、对接广告位。

技术实现

  1. 项目使用flutter开发，支持android和ios。

  2. 请求框架使用dio。

  3. 应用更新使用ota_update。

  4. h5页面使用webview_flutter加载。

  5. 接口请求的request、response数据均加密，使用encrypt、crypto库实现。

  6. 使用Provider实现全局状态管理和UI更新。
